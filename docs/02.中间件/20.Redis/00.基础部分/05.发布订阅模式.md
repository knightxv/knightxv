## Redis 的发布和订阅

## 1、什么是发布和订阅

-   Redis 发布订阅 (pub/sub) 是一种消息通信模式：发送者 (pub) 发送消息，订阅者 (sub) 接收消息。
-   Redis 客户端可以订阅任意数量的频道。

## 3、发布订阅命令行实现

### 1、打开一个客户端订阅 channel1

```sh
SUBSCRIBE channel1
```

### 2、打开另一个客户端，给 channel1 发布消息 hello

```sh
publish channel1 hello
```

-   返回的 1 是订阅者数量
